# Networking Basics

This repository contains introductory exercises and projects focused on fundamental networking concepts.

## Project Structure

- **basics_0/** - Core networking fundamentals including:
  - OSI model
  - Network types
  - MAC and IP addressing
  - TCP and UDP protocols
  - Port operations
  - Network connectivity testing

## Author

Holberton School Project

## Learning Objectives

- Understand the OSI model and its layers
- Differentiate between LAN and WAN networks
- Learn about MAC and IP addresses
- Explore TCP and UDP protocols
- Work with network ports and connectivity
